# Gerenciamento de Matrículas

Projeto feito com Django 3.2 (Python 3.8.5)

## Desenvolvimento

### Ambiente Virtual

```terminal
# criar um ambiente virtual
python -m venv .venv

# ativar o ambiente virtual
pipenv shell
```

## Dependências

```terminal
pipenv install
```

## Migrações

```terminal
python manage.py migrate
```

## Servidor

```terminal
python manage.py runserver
```
